# Win2x (Frontend + Backend)

This repository contains:
- `win2xnext` frontend (Next.js)
- `win2xnext/backend` backend (Node.js + WebSocket + MongoDB)

## Environment Files

Frontend:
- Copy `.env.example` to `.env`

Backend:
- Copy `backend/.env.example` to `backend/.env`


## Local Run

Frontend:
```bash
npm install
npm run dev
```

Backend:
```bash
cd backend
npm install
npm run dev
```
